# Modern Image Slider

A clean and modern image slider built using **HTML, CSS, and JavaScript**.  
This project features smooth animations, interactive buttons, and a visually appealing UI.

---

## Features

-  Smooth sliding animation
- ⏯ Auto-slide functionality
- ⏸ Pause on hover
-  Stylish "See More" button with hover effect
-  Background image transitions
-  Lightweight and fast
-  Basic responsive support

---

##  Technologies Used

- HTML5
- CSS3 (Flexbox, Animations)
- JavaScript (DOM manipulation)


---

## How to Use

1. Clone the repository:

git clone https://github.com/your-username/image-slider.git


2. Open the project folder

3. Run `index.html` in your browser
